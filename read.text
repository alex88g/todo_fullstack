
## Deployment instruktioner för Render

1. **Skapa PostgreSQL database på Render:**
   - Gå till Render dashboard
   - Klicka "New +" → PostgreSQL
   - Välj plan (Free tier finns)
   - Namnge databasen "todo-db"
   - Klicka "Create Database"

2. **Deploy backend:**
   - Klicka "New +" → Web Service
   - Anslut till ditt GitHub repo
   - Namn: "todo-backend"
   - Root Directory: "backend"
   - Build Command: `npm install`
   - Start Command: `node server.js`
   - Lägg till miljövariabler:
     - `DATABASE_URL`: din Render PostgreSQL connection string
     - `NODE_ENV`: production

3. **Deploy frontend:**
   - Klicka "New +" → Static Site
   - Anslut till ditt GitHub repo
   - Namn: "todo-frontend"
   - Root Directory: "frontend"
   - Build Command: `npm run build`
   - Publish Directory: "dist"
   - Lägg till miljövariabel:
     - `VITE_API_URL`: din backend URL (t.ex. https://todo-backend.onrender.com)

4. **Initiera databasen:**
   - Använd pgAdmin eller Render's webbaserade PostgreSQL editor
   - Kör SQL-koden från `database.sql`

Kom ihåg att exkludera `node_modules` och `.env` filer när du skapar ZIP-filen!
